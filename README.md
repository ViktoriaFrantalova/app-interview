# app-interview

This application contains two separate apps for the frontend and backend part.

## Start

### Frontend

Open bash terminal and use this command. You'll dowlound all packages.

```
yarn install / yarn
```

Run frontend application.

```
yarn start
```

#### Unit testing

Use command:

```
yarn test
```

### Backend

Open bash terminal and use this command. You'll dowlound all packages.

```
yarn install / yarn
```

Run backend application.

```
yarn start
```
